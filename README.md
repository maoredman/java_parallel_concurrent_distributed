## Subdirectories
#### `parallel_streams`
Used the Stream API provided by Java 8 to implement filter-map-reduce operations. Methods used include:
* `parallel()` to speed up computation
* `filter()`
*  `map()` methods
*  reduction methods such as `average()`, `count()`, `max()`
